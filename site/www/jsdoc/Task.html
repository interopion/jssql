Class: Task

Task

The Task class can be used to create Tasks suitable for adding into transactions. The user is only required to provide "execute" and and "undo" implementations. However additional properties might be defined:

new Task(options)

Parameters:
Name Type Description
options Object
Properties
Name Type Description
name String Optional. The name of the task. Defaults to "Anonymous task".
timeout Number Optional. The timeout for the task. If not provided, the global timeout option of the transaction will be used.
execute function The function that will be invoked as the task worker. If this is not provided, an empty function is used and a warning is generated to the console.
